# Ad-Campaign-Analytics-Dashboard-Looker-Studio-PostgreSQL
Marketing analytics dashboard with SQL-powered KPIs (CPC, CTR, CPM, ROMI) and interactive visualizations in Looker Studio.

## 📊 Project Overview
End-to-end marketing analytics dashboard built using SQL and Looker Studio.
Analyzes ad campaign performance across multiple channels with interactive filters.

## 🛠️ Tools & Technologies
- **SQL / PostgreSQL** — data extraction and KPI calculation
- **Looker Studio** — interactive dashboard and visualizations

## 📈 Key Metrics
- Ad Spend, CPC, CPM, CTR, ROMI

## 📁 Files
- `Ad-Campaign-Analitycs.sql` — SQL queries for KPI calculations
- `Facebook_&_Google_Marketing_Campaign_Report.pdf` — dashboard report

## 🔗 Dashboard Link
[View Dashboard](https://lookerstudio.google.com/s/jEr-DPEc-ls)

## 🔍 Key Findings
- **Trendy** campaign achieved the highest ROMI (191) with relatively low spend ($199K)
- **Discounts** campaign had the highest CTR (2.57%) at the lowest CPC ($8.88)
- **Expansion** was the biggest spender ($11.3M) but delivered average ROMI (119)
- **Promos** outperformed similar-budget campaigns with ROMI of 144
- Campaign activity peaked mid-2021 with 7 concurrent campaigns running
